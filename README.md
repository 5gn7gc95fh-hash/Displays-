# Inventario de Displays

Versión mejorada para GitHub Pages.

Funciones:
- Inventario
- Ventas
- Ventas por cobrar
- Garantías/fallas
- Uso interno
- Bitácora
- Por pedir
- Exportar respaldo JSON
- Importar respaldo JSON compatible
- Bloqueo de zoom en iPhone/Safari
- Logo y PWA

## Subir a GitHub Pages

1. Descomprime el ZIP.
2. Sube los archivos sueltos al repositorio.
3. Ve a Settings > Pages.
4. Source: Deploy from a branch.
5. Branch: main / root.
6. Guarda.


## Respaldo total

El botón **Respaldo total** guarda inventario completo, ventas, garantías/fallas, uso interno, bitácora, metadatos y una copia del almacenamiento local de la app.
